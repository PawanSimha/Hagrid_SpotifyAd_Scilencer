System lifecycle

How Hagrid achieves precision silence.

01

Event interception

Hagrid registers a lightweight on-device notification listener to track live background media stream metadata without polling or battery overhead.

02

Signature parsing

The local matching engine instantaneously cross-references tracking parameters and duration signatures without network round-trips or cloud dependencies.

03

Surgical muting

The moment an ad state is validated, the service flags Android's native AudioManager to drop stream volume to 0%, instantly restoring it when the next valid music index begins.
